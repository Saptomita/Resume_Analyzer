# 🚀 AI Resume Analyzer

An intelligent web application that analyzes resumes and provides ATS (Applicant Tracking System) scores, skill matching, and improvement suggestions based on job descriptions.

## 🌐 Live Demo

👉 (https://rainternship.netlify.app/)

## 📌 Features

* 📄 Upload resume (PDF)
* 🤖 AI-based skill detection
* 🎯 Job description matching
* 📊 ATS Score calculation
* ⚠️ Missing skills identification
* 📂 Resume section analysis
* 📥 Downloadable report (PDF)

## 🛠️ Tech Stack

**Frontend:**

* React.js
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Libraries & Tools:**

* pdf-parse (Resume parsing)
* multer (File upload)
* natural (NLP processing)
* jsPDF (Report generation)

**Deployment:**

* Netlify (Frontend)
* Render (Backend)


## ⚙️ How It Works

1. Upload your resume
2. Paste job description
3. Click **Analyze Resume**
4. Get:

   * ATS Score
   * Job Match Score
   * Detected Skills
   * Missing Skills
   * Resume Feedback
   * Download the report
